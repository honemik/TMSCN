革蘭氏陽性

// 此篇僅針對臨床上常見之致病菌作介紹

球菌 Cocci

Gram Positive Cocci

  • Catalase + ( Staphylococci )
    • Coagulase +
      • S. aureus
    • Coagulase -
      • S. epidermidis
  • Catalase - ( Streptococci )
    • alpha hemolysis
      • optochin sensitive ( pneumoniae )
      • optochin resistant ( viridans )
    • beta hemolysis
      • GAS ( pyogenes )
      • GBS ( agalactiae )
    • gamma hemolysis
      • enterococcus

葡萄球菌屬 Straphylococci

簡介

  • 葡萄樣子 ( grapelike ) /不規則聚集的球菌
  • 直徑 0.5 - 1.5
  • 不會移動 ( nonmotile )
  • 兼姓厭氧菌

金黃色葡萄球菌 Staphylococcus aureus

特性

  • 在綿羊血培養基中，會出現溶血特性，在菌落周圍形成一圈似光環的區域 ( 全溶血 )
  • 不透明的珍珠狀圓型菌落
  • 生長最適溫度為攝氏 37 度，但能忍受高溫、高鹽以及極端的 pH 值環境

毒力因子

結構 structure

  1. 莢膜 ( Capsule )：抑制吞噬作用
  2. 黏液層 ( Slime Layer )：抑制吞噬作用、有黏著蛋白 ( Adhesion Proteins )能黏附於人體組織
  3. 肽聚醣 ( Peptidoglycan )：誘導白血球不正常釋放細胞激素 ( Cytokines )
  4. 磷壁酸 ( Teichoic Acid )：金黃色葡萄球菌細胞壁特有，連接纖連蛋白 ( Fibronectin )
  5. Protein A：中和抗體，結合免疫球蛋白 ( IgG1、IgG2、IgG4 )上的FC-region
  6. Coagulase：使血液凝集

毒素 Toxins

  1. 細胞毒素 ( Cytotoxins )
    • 攻擊宿主細胞膜
    • 對於紅血球、白血球、血小板、巨噬細胞、纖維母細胞有毒性
    • 分類
      • α toxin：可見於幾乎所有金黃色葡萄球菌，會損害組織
      • β toxin：一種鞘磷脂酶 ( Sphingomyelinase C )，能水解磷脂質，造成細胞膜崩解，若發生於紅血球，即為全溶血
      • δ Toxin：非專一性降解宿主細胞膜，似界面活性劑
      • γ Toxin：多分子聚合體，能在細胞膜形成孔洞，進而使細胞破裂，部分菌種無此毒素
      • Panton-Valentine ( P-V )：一種獨立蛋白，功能同 γ Toxin，亦非所有金黃色葡萄球菌皆有
  2. 脫皮毒素 ( Exfoliative Toxins )
    • Serine Protease ( Serine 分解酶 )
    • 可分解真皮層與表皮層間的連結蛋白 ( Lyze desmoglein 1 )
    • 引起脫皮症候群 ( Scalded Skin Syndrome )，患部不一定採到細菌 ( 隨表皮脫落 )
    • ETA ( Exfoliative A )
      • 熱穩定
      • 源於噬菌體插入到細菌的基因轉錄轉譯合成
    • ETB ( Exfoliative B )
      • 熱不穩定
      • 源於質體
  3. 腸毒素 ( Enterotoxins )
    • Enterotoxins A 最為常見
    • 常見的食物中毒致病原
    • 熱穩定，加熱後僅細菌死亡，100℃維持三十分鐘以上，毒素才會失去活性
    • 一種超級抗原 ( Superantigen )：會異常活化免疫細胞，使T 細胞增生，造成全身免疫反應
    • 作用在腸道，刺激肥大細胞釋放發炎介質後，引起腸蠕動增加、噁心、嘔吐、水分流失等
  4. 中毒性休克綜合症毒素 ( TSST-1, Toxic Shock Syndrome Toxin-1 )
    • 一種超級抗原，能引發全身性休克反應，且通常無外傷
    • 90%的TSS 由金黃色葡萄球菌所引起，有多起臨床案例是因為使用衛生棉條而感染到特殊金黃色葡萄球菌，引發嚴重休克

酵素

  1. 凝固酶 ( Coagulase )
    • 把可溶性的 Fibrinogen 轉換為不可溶的 Fibrin，以形成凝集 ( Clump )
    • Bound Form 黏附於細菌上 / Free Form 分泌至細胞外
  2. 玻尿酸分解酶 ( Hyaluronidase )：水解結締組織
  3. 纖維蛋白溶解酶 ( Fibrinolysin )：水解纖維蛋白凝塊
  4. 脂酶 ( Lipase )：水解脂質
  5. 核酸酶 ( Nuclease )：水解核酸
    • 補充分裂時所需的遺傳物質原料
    • 協助分解嗜中性球為捕捉細菌而釋放的網子 ( Extracellular net )

臨床疾病

  1. 局部皮膚性感染 ( Cutaneous Infections )：可由傷口、毛囊、腺體入侵
  • 毛囊炎 ( Folliculitis )：僅限於毛囊附近，治療後少有併發症產生
  • 癤 ( Furuncle )：由毛囊炎發展成的膿腫
  • 癱 ( Farbuncle )：由大範圍癤的不同毛囊串起所形成的病變
  • 膿疱 ( Impetigo )：小火山狀的化膿即紅腫，常見於新生兒
  1. 系統性感染：通過血液感染
    • 骨髓炎 ( Osteomyelitis )：多發炎和發膿於骨骺端 ( Epiphysis )，引發疼痛甚至骨頭斷裂
    • 敗血性關節炎 ( Septic Arthritis )：化膿物質堆積於關節空隙中
    • 菌血症 ( Bacteremia )：細菌進入血液循環中，進而可能轉移至其他器官，如心臟
    • 心內膜炎 ( Endocarditis )：多為急性心內膜炎，常見於術後感染
    • 肺炎 ( Pneumonia )：多見於感染老人及幼童，如若惡化會導致膿胸 ( Empyema )，更甚者引發敗血性休克 ( Septic Shock )，死亡率極高

流行病學

  • 20％～60％健康成人的鼻腔前側、皮膚、鼻咽、腸道帶有金黃色葡萄球菌
  • 經由衛生較差的環境、組織受傷、糖尿病、免疫缺失、接觸傳染等途徑傳染給他人
  • 家人或同居者為攜帶者，感染機率大幅提升
  • 抗藥性問題：抗藥性金黃色葡萄球菌 ( MRSA, Methicillin-Resistant S. aureus )
    • 抗生素的濫用，讓細菌對甲氧西林 ( Methicillin ) 有抗性
    • 甲氧西林 ( Methicillin ) 為半合成的盤尼西林 ( Penicillin )
    • 作用機制
      • 細菌的mecA 基因可轉譯出PBP ( Penicillin Binding Protein ) ，強化肽聚糖細胞壁的共價鍵結
      • β-Lactum 抗生素 ( 如Penicillin ) 與PBP 親和力強，能被送進細胞，抑制細胞壁合成而殺菌，其與Penicillin 親和力差，此類抗生素便無法消滅MRSA
  • MRSA 對台灣院内感染之影響
    • 早期高達80％，現今下降到約60％，但仍然是個嚴重的問題
    • 醫院型MRSA ( HA-MRSA, Hospital-associated MRSA )
      • 院内發生
      • 具多重抗藥性 ( Multidrug-resistant )，因醫院病人大多接受過各種抗生素治療
    • 社區型MRSA ( CA-MRSA, Community-associated MRSA )：近二十年發現
      • 特殊型別：SCCmecIV 會產生 P-V Leukocidin Toxin
      • 除局部皮膚感染，也經常引起嚴重肺炎 ( Severe Pneumonia )，拉高致死率
      • 基因有地域性，不同的國家流行的型別不同 ( USA：USA300，Asia：ST59 )
      • 較不具抗藥性，β-Lactams 以外的其他抗生素仍對CA-MRSA 有作用

治療和預防

實驗室診斷

顯微鏡觀察

革蘭氏染色呈現陽性，不規則紫色球狀群聚

細菌培養 Culture

產生完全溶血性 ( β-Hemolytic ) 菌落 ( Colony )

細菌鑑定 Identification

Staph Tri-plate，若三者皆呈陽性，此菌爲S.aureus 的機率很高

  1. DNase Agar：是否產生熱穩定的核酸分解酶 ( Heat-stable Nuclease )
    • 培養基含聚合DNA，與甲基綠結合，呈藍綠色
    • 篩選出的細菌有DNA 水解酶，能將培養基中的分解成小片段，藍綠色消失
  2. Mannitol Salt Agar：檢測是否對甘露醇產生發酵作用 ( Mannitol Fermentation )
    • 培養基高鹽 ( 7.5% NaCl )且含甘露酸 ( Mannitol )
    • 篩選出的細菌能夠耐鹽，同時將甘露酸發酵，而發酵的酸性產物使酚紅從紅變黃
  3. Mueller-Hinton Agar：檢測對新生黴素的耐受性 ( Novobiocin Susceptibility )

快速凝集檢測

檢驗是否含有Coagulase 和Protein A

鏈球菌屬 Streptococcus

特徵

  • 革蘭氏陽性 ( 紫色 )、觸酶 ( Catalase ) 陰性
  • 成串或成對排列
  • 不會移動 ( Nonmotile )、不產孢子
  • 對環境要求高，需要豐富營養物質培養，一般常用新鮮血盤或血清
  • 鏈球菌屬共生在人體口腔中占大比例，大部分無害，少部分在特殊情況可能引起疾病
  • 部分菌種能夠存活於充滿CO2 的環境中

分類

血清學：古早的分類方式，利用細菌表面的醣蛋白來分類，目前主要用於全溶血的細菌內

溶血反應 ( Hemolysis Reactions )：臨床上常用的分類方式

  • 半溶血 ( α-Hemolysis )：血基質氧化會形成草綠色菌落，再以Optochin Test 細分
    • 肺炎鏈球菌 ( S. pneumoniae )：Optochin 陽性
    • 草綠鏈球菌 ( Viridans streptococci )：Optochin 陰性，其他非肺炎鏈球菌的菌種統稱
  • 全溶血 ( β-Hemolysis )：可再以血清學分類，針對細胞壁上特殊醣類C-Carbohydrates 的組成，利用Bacitracin Test 檢測，依照Lancefield 分類法
    • GAS ( Group A Streptococci )：化膿性鏈球菌 ( S. pyogenes )
    • GBS ( Group B Streptococci )：無乳性球菌 ( S. agalactiae )
  • 不溶血 ( γ-Hemolysis )：Enterococcus ( 腸球菌 )，已從鏈球菌屬中獨立

生化反應：主要適用於α-和γ-Hemolysis，現由蛋白質圖譜鑑定法取代

化膿性鏈球菌 Streptococcus pyogenes

特性

  • 屬於Group A Streptococci ( GAS )，β-Hemolysis
  • 球菌，直徑1～2μm，聚集成短鏈狀，菌落周圍有明顯的溶血圈
  • 在營養環境下生長較佳，但高濃度葡萄糖反而會抑制其生長
  • 噬肉菌其中一種

感染

  • 主要造成細菌性咽頭炎 ( Bacterial Pharyngitis )
  • 併發症
    • 壞死性筋膜炎 ( Necrotizing Fasciitis )，故又稱作噬肉菌 ( Flesh-eating Bacteria )
    • 嚴重的系統性感染

抗原性結構

  • 細胞壁上C-Carbohydrate：屬於GAS，由Rhamnose 和N-Acetylglucosamine 組成
  • 表面蛋白：M protein 可做進一步的菌株分類
    • C 端黏著在細菌菌體上高度保留，N 端變異度高，產生不同型別，造成不同症狀
    • 由Emm Genes 調控，與細菌的致病性有關，如Class I 與風濕熱 ( Rheumatic Fever ) 有關

毒力因子

結構

  • 莢膜 ( Capsule )：由玻尿酸 ( Hyaluronic Acid ) 構成，能偽裝成人體細胞，降低抗原性
  • M Proteins：抑制補體中C3b 的結合，進而阻止吞噬作用
  • M-Like Proteins：結合抗體的Fc 片段，阻斷補體活化並減少C3b 結合的機會
  • 黏附宿主細胞：LTA ( Lipoteichoic Acid )
  • 幫助細菌進入深層組織：M Proteins、F Proteins ( 老師說不要理他 )

毒素

外毒素

  1. 化膿性外毒素 ( Streptococcal Pyogenic Exotoxin ) / 發紅毒素 ( Erythrogenic Exotoxin )
    • 熱不穩定型毒素 ( Heat Labile Toxins )，有不同的型別，如SpeA-C、SpeF，經由溶原性菌株 ( Lysogenic Strains )製造，溶原性是指噬菌體的核酸會被插在細菌染色體上
    • 本身是種超級抗原 ( Superantigen )
    • 造成壞死性筋膜炎 ( Necrotizing Fasciitis )、鏈球菌毒素的休克反應和猩紅熱患者的紅疹
  2. 溶血素 ( Hemolysins/Streptolysins )
    • Streptolysin S ( SLS )：氧氣穩定型 ( Oxygen Stable )，能和紅血球、白血球、血小板等結合並破壞宿主細胞，造成全溶血，或是藉由分解Lysosome 導致吞噬細胞死亡 ( β-Hemolysis )
    • Streptolysin O ( SLO )：氧氣不穩定型 ( Oxygen Labile )，具有抗原性，會導致病患產生ASO ( Anti-streptolysin O Antibodies )，臨床上以檢測此抗體來判斷是否感染過GAS，但因皮膚上有膽固醇能抑制ASO，故不會呈現此抗體；較常見是咽喉炎感染部位有ASO

酵素

細菌表面：C5a Peptidase：能降解C5a

胞外

  1. 鏈激酶 ( Streptokinase A & B )：溶解血栓，現被製成藥物
  2. 玻尿酸酶 ( Hyaluronidase )：分解結締組織
  3. DNase：水解DNA，降低膿瘍中物質的黏度

流行病學

  • 化膿性鏈球菌感染於健康兒童或成人的口咽部 ( 無症狀帶原者 )
  • 飛沫式感染，主要感染群體為兒童，由皮膚或是咽喉感染
  • 若未接受治療可能導致全身性感染或產生後遺症
  • 未接受治療之患者可能會產生M Protein 抗體 ( 長期免疫 )

臨床疾病

喉嚨感染：鏈球菌性咽炎 Streptococcal Pharyngitis

  • 接觸病原後2～4 天出現症狀
  • 症狀包括喉嚨痛，發燒，全身不適 ( Malaise )和頭痛
  • 相較於病毒性咽炎，舌苔會出現反白，咽喉部有明顯紅腫
  • 併發症：猩紅熱 ( Scarlet Fever )
    • GAS 攜帶有噬菌體的熱源毒素 ( Pyrogenic Toxin )核酸 ( Prophage )
    • 全身紅斑 Diffuse Erythematous Rash ( 較易觀察 )
    • 草莓舌 Strawberry Tongue ( 較易觀察 )
    • 皮斯塔線 Pastia Line ( 較難觀察 )
    • 嘴巴周圍泛白 Circumoral Pallor ( 較難觀察 )

皮膚感染

引起皮膚感染的菌株與引起咽炎的不同

  1. 膿疱病 ( Pyoderma )
    • 因皮膚接觸帶原者或病原體而感染
    • 皮膚破裂 → 充滿膿液的囊泡 ( 膿皰 )→ 膿皰破裂 → 結痂
    • 局部淋巴結腫大但無全身感染徵象
  2. 丹毒 ( Erysipelas )
    • 急性皮膚感染伴有疼痛和全身性的症狀
    • 常見發病於四肢與臉部
    • 感染和未感染的皮膚之間界線明顯
    • 好發於老年人或幼兒
  3. 蜂窩組織炎 ( Cellulitis )
    • 感染皮膚和深層皮下組織
    • 感染和未感染的皮膚之間界線模糊
  4. 壞死性筋膜炎 ( Necrotizing Fasciitis )
    • 感染皮下組織深處，沿著筋膜擴大，破壞肌肉與脂肪組織
    • 破壞皮膚 → 蜂窩性組織炎 → 壞疽 ( Bullae/Gangrene )→ 全身性症狀
    • 因毒性造成多重器官衰竭甚至死亡
    • 需積極治療，透過清創手術 ( Surgical Debridement )去除已感染之組織

系統性感染

  1. 菌血症 ( Bacteremia )
  2. 肺炎 ( Pneumonia )
  3. 產褥熱 ( Puerperal Sepsis )： 以前衛生條件較差時較常發生
  4. 鏈球菌中毒性休克 ( Streptococcal Toxic Shock Syndrome )
  • 多數有壞死性筋膜炎或是菌血症
  • 菌株具有特殊黏多醣莢膜 ( Mucopolysaccharide Capsules )與外毒素 ( SpeA and SpeC )
  • 外毒素引起之系統性反應，造成中毒性休克

併發症

  1. 風濕熱 ( Rheumatic Fever )：咽喉炎之併發症
    • 菌株帶有特別的M Protein 抗原，交叉辨識到自身細胞，導致產生會攻擊自體細胞之抗體
    • 發炎易轉移至心臟、血管或是關節處
  2. 急性腎絲球腎炎 ( Acute Glomerulonephritis )：皮膚感染之併發症
    • 由腎型M Protein 引起
    • 引發腎炎、血壓升高、偶爾會有心臟衰竭，可能演變成慢性腎衰竭

實驗室診斷

  • 革蘭氏染色：陽性，會與白血球結合，成對或成鏈狀
  • 細菌培養：從口咽後部 ( Posterior Oropharynx )採檢，需要長時間培養
  • 血清學測試：針對併發症，做ASO ( Anti Streptolysin O )檢驗來驗證是否為細菌性感染造成

乳鏈球菌/B 型鏈球菌 ( Streptococcus agalactiae )

GBS 唯一β-Hemolysis 的細菌，比起GAS 菌落較大，溶血性沒那麼強

抗原性結構

  • 細胞壁上C-Carbohydrate：屬於GBS：由Rhamnose、N-acetylglutcosamine、Galactose 組成
  • 莢膜上的多糖：共有9 種，因種類 ( Ia、Ib、II～VIII )而異
  • 表面蛋白：最常見的為C-抗體

毒力因子

  • 莢膜：干擾吞噬作用
  • 唾液酸 ( Siliac acid )：Ia、Ib、II 型帶有，能抑制補體Alternative pathway

臨床疾病

新生兒感染

引發敗血症 ( Septicemia )和腦膜炎 ( Meningitis )

Early-OnsetLate-Onset
感染時間剛出生出生後一週至三個月
引發病症菌血症、肺炎、腦膜炎菌血症、腦膜炎
癒後15～30%出現神經系統後遺症 ( Neurologic Sequelae )25～50%出現神經系統併發 ( Neurological complication )

診斷：檢查腦脊髓液

成人感染

懷孕中的婦女：子宫内膜炎、泌尿生殖道感染，以抗生素治療，癒後良好

其他：伺機性感染，引發菌血症、肺炎與骨頭、關節、皮膚、軟組織感染，尤其是身體虛弱者如高齡族群，常伴隨高死亡率

流行病學

  • GBS 屬於存在於人體下消化道與泌尿生殖道裡的正常菌種
  • 可能於自然產過程中，傳染給新生兒，故常使用分娩期抗生素預防
  • 孕婦中成人GBS 感染的風險比男性和非孕婦中的更大

實驗室診斷

  • 針對新生兒感染：新生兒感染乙型鏈球菌 ( GBS )會導致腦膜炎。因此直接採腦脊髓液做格蘭氏染色，因腦脊髓液通常無菌，若出現結果為紫色小球沉澱狀，通常就是GBS 感染
  • 針對婦女陰道採檢，多使用PCR

細菌培養：臨床少用

  • 抗Bacitracin
  • CAMP Factor 陽性，其可加強溶血
  • 能分解Hippurate

草綠鏈球菌 Viridans streptococci

特性

  • 由一群α-Hemolytic 與γ-Hemolytic 鏈球菌組成
  • 肺炎鏈球菌 ( S. pneumoniae )是該種類中最毒者
  • 大部分細菌常見分佈於口腔、口咽 ( Oropharynx )、消化道 ( GI )、生殖道 ( Genital Tract )、皮膚
  • 平時不具侵略性，多為伺機性感染，例如：洗牙、清創造成的外傷感染

分類

Mitis Group：引起亞急性心內膜炎 ( Subacute Endocarditis )

  • 相比於急性的金黃色葡萄球菌感染，較不凶險
  • 透過血液循環至心臟，黏附於心內膜後，增殖形成一層膜 ( Vegetation )
  • 除了要配合抗生素治療外，也需要手術清除

Mutans Group：將醣類轉為酸，使珐瑯質損傷，造成蛀牙，最主要的菌種為S. mutans

肺炎鏈球菌 Streptococcus pneumoniae

特徵

  • 橢圓形的革蘭氏陽性菌，屬於Mitis Group
  • 多醣莢膜 ( Polysaccharide Capsule )厚，依據其中的醣類組成能分辨菌株，目前有90 種血清型
  • 對生長環境要求很高，故不容易在體外被發現，加入5～10％的二氧化碳可幫助成長

致病機轉

  • 沒有明顯的外毒素
  • IgA Protease：分解IgA 蛋白 ( 黏膜上的抗體 )
  • Pneumolysin：結合在宿主細胞膜上的膽固醇，並造成穿孔
  • Adhesin：黏附素，可使細菌附著在宿主細胞上，幫助Colonization

流行病學

  • 好發於秋冬，容易感染60 歲以上的老人、6 個月到4 歲大的嬰幼兒與免疫功能低下者
  • 屬於人體中的正常菌叢，因此感染多是內生性 ( Endogenous )的，在人體免疫力降低時才作用

臨床疾病

  • 大葉性肺炎 ( Lobar Pneumonia )：人類大約60～70％的肺炎是由該菌種引起
    • 前期常被視為一般感冒，後期的症狀往往是突發性的 ( Abrupt )
    • 常伴隨高燒、胸痛，嚴重時會咳血
    • 容易在感染流行性感冒病毒後免疫力低下時再被肺炎鏈球菌感染，因此疫苗常常搭配一起打
  • 鼻竇炎 ( Sinusitis )
  • 中耳炎 ( Otitis Media )：好發於五歲以前的孩童
  • 腦膜炎 ( Meningitis )：稍罕見，有較高的死亡率，且可能造成嚴重神經缺損
  • 菌血症 ( Bacteremia )：發生於25～30％的肺炎球菌性肺炎 ( Pneumococcal Pneumonia ) 患者，而可見於大約80％的Meningitis 患者中

實驗室診斷

  • 取病患的痰（Sputum Specimen）做革蘭氏染色，會出現狹長、成對且被沒染色的莢膜所包圍的菌種。但由於痰中可能出現很多種細菌，因此失敗率頗高（下圖最左）
  • 莢膜腫脹試驗（Quellung Reaction）：菌落加入血清後，細菌會產生大量的莢膜，可觀察顯微鏡下是否有莢膜腫脹（Swelling）的微生物
  • 腦膜炎感染：採病患的脊髓液（CSF）做血清學檢測
  • 分辨肺炎鏈球菌與其他α-Hemolytic Species 的方法
  • 菊糖發酵（Inulin Fermentation）：肺炎鏈球菌可發酵菊糖，使試劑由紅轉黃（下圖右中）
  • 奧普托辛測試（Optochin Test）：肺炎鏈球菌對Optochin 較敏感
  • 膽汁溶解度測試（Bile Solubility Test）：肺炎鏈球菌會被膽鹽溶解（下圖最右之右管）

預防與治療

  • 抗生素治療：大多以Penicillin 治療，但近年抗藥性越來越明顯
  • 施打疫苗：通常在即將進入秋冬季時施打以莢膜製作的疫苗，目前的疫苗主要有以下兩種
    • PPV（Pneumococcal Polysaccharide Vaccine）：可抵抗23 種血清型，適用於2 歲以上的人類
    • PVC13：可用於兩個月到五歲大的嬰幼兒，使用13 個肺炎鏈球菌的抗原製成

腸球菌 Enterococcus

原本被歸類於D 型鏈球菌（Group D streptococci），在1984 年後重新分支出來獨立

外型：革蘭氏陽性（紫色），成對或呈短鏈狀，型態類似肺炎鏈球菌（Streptococcus pneumonia）

特性：環境適應力強，耐酸鹼、高溫、高鹽和高膽鹽，類似金黃色葡萄球菌

致病力

  • 能黏附到組織、管路上，產生生物膜（Biofilm），不容易治療
  • 擁有先天抗藥性，能抵抗很多現行使用的抗生素，難以根治
  • 能透過水平式的基因轉移獲取並散播抗藥性基因，而能對抗Aminoglycoside、Vancomycin

流行病學

  • 與相關疾病人體正常的腸內菌，一般存在於大腸中
  • 因免疫力下降，或使用其他儀器管線，如導尿管而感染，為院內感染的主要細菌之一
    • 泌尿道感染（UTI, Urinary Tract Infections）
    • 腹膜感染（Peritoneal Infections）
    • 菌血症（Bacteremia）
    • 心內膜炎（Endocarditis）
  • 在過去被認為是不重要的致病菌，但現在因抗藥性強，威脅性升高
    • 因其帶有能更改肽聚糖側鏈最後兩個胺基酸的基因，而能抗Vancomycin
    • 菌株抗Vancomycin 的能力在十年間從不到20％到現在已經40％，且為全球性趨勢
    • 目前並不清楚為什麼會短期內在全世界就有那麼多抗Vancomycin 的菌株

實驗室診斷

  • 不溶血，且其能在Blood Agar 中生長得很好（很好養也好分離）
  • 對Optochin 有抗性
  • 不會被膽汁分解
  • 會產生一種酵素PYR（Pyrrolidonyl Arylamidase）
