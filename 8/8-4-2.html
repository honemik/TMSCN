縮小包膜絛蟲 Hymenolepis diminuta

又稱鼠絛蟲，在老鼠身上為世界性分布，很少感染人，人類為偶然宿主 在西班牙夢幻群島 ( Canary islands )曾有人感染 感染型為擬囊尾蟲 ( Cysticercoid )，高腳杯狀，前端膨大處會發展成頭節

感染

避免吃到中間宿主 ( 蟑螂、甲蟲 )，並防治鼠類

型態

頭節 Scolex

  • 四個吸盤
  • 具有退化的額突 ( Rostellum )，僅有一點突起的痕跡
  • 沒有吻鉤 ( Hooklets )

成熟體節 MP, Mature Proglottids

  • 總生殖孔開口在體節同一側
  • 共三顆睪丸：靠近生殖孔處一顆，另一側則有兩顆
  • 卵巢在體節中間，呈現似花瓣狀的分裂
  • 通常先找到生殖孔，再判斷睪丸的位置

受孕體節 GP

  • 子宮囊形狀像扭轉的麻花狀

蟲卵

  • 中大型蟲卵72 ( 56～86 )μm
  • 形狀：微卵圓型而近似圓形，像是荷包蛋
  • 卵殼為深棕色，具有放射狀的花紋，但比較小，不明顯
  • Taenia spp.是「胎蟲被殼」呈現放射狀紋，並非卵殼
  • 蟲卵外圈有一圈紅色折光 ( 可用來區分蟲卵和雜質—蟲卵才會有折射光，因為蟲卵中含水 ) ※若背景較淺時，紅色折光便容易觀察到，每種蟲卵都有機會看到 ( 薑片蟲最常觀察到 )
  • 蟲卵可觀察到三對剪刀，但並非每一個蟲卵都可以觀察到。但只要能在蟲卵中觀察到有一對剪刀，便可判斷此蟲卵屬於圓葉目絛蟲的蟲卵

生命週期

  1. 卵被中間宿主 ( 蟑螂、甲蟲以及其他節肢動物 )吃掉 → 在中間宿主體內發育成為感染型的Cysticercoid → 中間宿主被天然宿主 ( 鼠類 )或偶然宿主 ( 人 )吃下 → 寄生在小腸 → 受孕體節脫落→隨糞便排出
  2. 中間宿主：蟑螂、甲蟲以及其他節肢動物
  3. 保蟲宿主：鼠類
  4. 一般來說，老鼠為正常最終宿主；人類為偶然宿主，不小心吃到中間宿主才會感染，縮小包膜絛蟲還是會在人體內發育成成蟲，但是無法存活超過七個月
  5. 實驗室培養 ( 台灣現在沒有 )：拿蟲卵去餵食甲殼類動物，再把甲殼類動物拿去餵食老鼠，並從老鼠體中拿出蟲體，取出受孕體節中的卵，重複進行這個循環

分布

臨床症狀

  • 蟲體較少時，幾乎無症狀
  • 蟲體數目過多時，容易造成消化不良、腹痛、腹瀉、厭食、暈眩、頭痛等症狀

治療

  • Praziquantel
  • 瀉藥

診斷

檢查糞便中是否有蟲卵、受孕體節

短小包膜絛蟲 Hymenolepis nana/侏儒絛蟲

感染

避免吃甲蟲 ( 多因落後國家生活需求、風俗民情、誤食 )、防治鼠類

型態

大小

  • 成蟲長約25～40 公分，寬0.5～1 公分
  • 約有200 多個體節，為寄生在人體內最小的絛蟲

頭節

  • 四個吸盤
  • 額突 ( Rostellum )可自由伸縮
  • 吻鉤 ( Hooklets )

成熟體節

  • 總生殖孔開口在體節同一側
  • 三顆睪丸 ( 先找到生殖孔，靠近生殖孔的有1 顆，遠離生殖孔的有2 顆 )
  • 中間有卵巢分裂似啞鈴狀

受孕體節

  • 只有子宮特別發達，其他器官退化
  • 子宮囊形如袋狀從中扭轉成二，蟲卵充斥在子宮內

幼蟲

囊尾蚴 ( Cysticercoid )和Hymenolepis diminuta十分相似

蟲卵

  • 中小型蟲卵30～60 × 16～34 μm ※比較：H. diminuta為中大型蟲卵
  • 內含六鉤幼蟲
  • 有檸檬狀小突起 ( 因為角度不一定看的到 )
  • 其兩極具有2～4 對極絲 ( 但只有新鮮蟲卵才看得到 )
  • 蟲卵殼內有一層膜 ( 老師比喻為茶葉蛋蛋殼剝開後可見的蛋膜 )會具有不同的形態，有時完好無缺，有時會呈現皺縮情形，故膜不是一個判別蟲卵的依據

生命週期

  • 中間宿主：榖類甲蟲，亦可不需中間宿主，因此會產生自體感染 ( Autoinfection )
  • 保蟲宿主：鼠類
  • 感染期：受孕蟲卵 ( Embryonated Ova )、 擬囊尾蟲 ( Cysticercoid )

正常生活史：

  1. 受孕蟲卵在糞便中
  2. 蟲卵被穀類昆蟲吃到，在體內發育為Cysticercoid
  3. 被感染的節肢動物被老鼠吃到或被人類誤食，進入腸道發育為成蟲

自體感染：

※狹義定義：寄生蟲在人體內需從蟲卵變成幼蟲再變成蟲

  1. 絛蟲的受孕體節脫落，受孕體節因摩擦而使裡面的蟲卵跑出來
  2. 蟲卵停留在人體腸道，並發育為幼蟲，在同一宿主內重新完成生活史

分布

全世界

臨床症狀

  • 蟲體小，故數量少時無症狀
  • 蟲體數目過多時會有消化道症狀 ( ex：消化不良、腹痛、腹瀉、厭食、暈眩、頭痛 )
  • 有些人感染會產生類似蕁麻疹的病症
  • 自體感染時，危害會比較嚴重

治療

  • 用Atabrine，要配合瀉藥，以排出蟲體
  • 不用Praziquantel，因為該藥會造成體節破裂，使蟲卵跑出來，造成Autoinfection

診斷

包膜絛蟲 Echinococcus species

常見有四大類，但是因E. vogeli與E. oligarthrus分布在台灣較為少見，故主要介紹 單胞絛蟲 ( E. granulosus )與多胞絛蟲 ( E. multilocularis )為主

分布最終宿主中間宿主
E. granulosus全世界狗有蹄類 ( Ungulates )
E.multilocularis北半球狐狸鼠類 ( Voles )

Echinococcus分布地區：只要有畜牧業都有可能有

單胞絛蟲 Echinococcus granulosus

感染

型態

  • 約3～6 mm ( 約在顯微鏡一百倍下即可見到 )
  • 只有四個體節：頭節、未成熟體節、成熟體節、受孕體節，各一個
  • 若是從生長點 ( 頸部 )長新體節出來，最後一節便會脫落，所以會一直維持四個體節
  • 有四個吸盤，頭部有兩圈吻鉤 ( Hooklets )

包囊蟲 ( Hydatid Cyst )

由蟲卵發育而成，可看作為單胞絛蟲的幼蟲，由外到內可以分為三層

  1. 固有層 ( Adventitial Layer )：由宿主細胞構成的纖維狀被膜 ( Fibrous Capsule )
  2. 角質層/肌肉層 ( Laminated Layer )：由寄生蟲細胞構成的纖維狀被膜 ( Fibrous Capsule )
  3. 生發層 ( Germinal Layer )： 可向內生成Daughter Cyst 與Hydatid Sand
    1. 子囊 ( Daughter Cyst )：又叫做生發囊 ( Brood Capsule )，其內層也是類似Germinal Layer 的構造，不斷向內形成Cyst，使Hydatid Cyst 越長越大
    2. 頭砂 ( Hydatid Sand )/原頭節 ( Protoscolex )：兩者的構造和功能相同，未來都會發育成頭節，但差異處在於兩者的生成位置不同
      • 頭沙直接生成於Hydatid Cyst 生發層的內側
      • 原頭節生成於Daughter Cyst 的內側

生命週期

  1. 中間宿主：有蹄類 ( 羊、豬、牛 )
  2. 保蟲宿主：犬科動物 ( 狗、狼、狐狸 )的小腸中
  3. 感染型：蟲卵
  4. 過程
    1. 有蹄類攝入蟲卵 → 蟲卵在有蹄類體內的各個臟器發育成包囊蟲 ( Hydatid Cyst ) → 食肉動物攝入，發育成成蟲
    2. 若人類誤食蟲卵，也會在體內各個臟器發育成Hydatid Cyst，造成囊蟲症 ( Echinococcosis )

分布

畜牧地區

臨床症狀

囊蟲症 ( Cysticercosis )

  • Hydatid Cyst 其餘的空間充滿著液體，裡頭含有會引起TH1 與TH2 免疫反應的因子 ( 不用背 )
  • 若Cyst 破裂 ( 如手術 )使裡面的過敏原流出，病患容易引發強烈的免疫反應而死亡

治療

  • Albendazole ( 對幼蟲有效 )、Praziquantel
  • 手術直接摘除 ( 過去作法 )
  • PAIR ( Percutaneous Aspiration, Injection and Reaspiration )
    1. 以針孔戳進去抽液體減壓 → 灌入食鹽水或10%福馬林殺死裡面的頭節 → 取出Hydatid Cyst
    2. 先用電腦斷層掃描得知cyst 大小，估計應抽出多少液體 ( 維持形狀 )跟注入多少鹽水
    3. 注入鹽水使得Cyst 內鹽水濃度大於20% ( 可殺死頭節 )
    4. 七到十分鐘後再抽出Cyst 內液體

※治療前一天和完成後的一個月內應以Albendazole 治療，以確保不再形成新的Hydatid Cyst

診斷

  • 抽Cyst 的液體看有沒有頭節 ( 危險作法，液體流出會引起嚴重免疫反應 )
  • Casoni Test：免疫診斷法 ( 0.1 c.c.的Hydatid Cyst Fluid 打入皮下 )，會有交叉反應
  • 用電腦斷層、超音波或X-ray 掃描

多胞絛蟲 Echinococcus multilocularis

感染

避免接觸狐狸、犬的排泄物

型態

生命週期

  1. 中間宿主：鼠類、人類 ( 打獵或做皮毛的人易感染 )
  2. 保蟲宿主：狗、狐狸
  3. 感染型：蟲卵
  4. 幼蟲形態：共尾囊蟲 ( Coenurus )長得像一大串葡萄的樣子，稱為Alveolar Hydatid
  5. 過程
    1. 老鼠吃掉蟲卵 → 在肝發育成幼蟲 → 狐狸攝取老鼠肝臟 → 在體內發育成蟲
    2. 人類因誤食吃到蟲卵後會在各個器官內發育為幼蟲

分布

北半球為主，因為終宿主多分佈在北半球

臨床症狀

Alveolar Hydatid Disease

  • 在內臟產生囊狀構造，稱作Coenurus
  • 多在肝臟發生，但也會在肺部和腦等

治療

Albendazole

診斷

  1. 抽取Coenurus 中的液體，檢測是否有頭節
  2. Immunologic test ※E.granulosus、E. multilocularis形態學上不易區分，假設蟲體都是四節時，可用以下方法診斷
  3. PCR
  4. Western Blot：單胞絛蟲在16～18 kDa 有diffuse band，多胞絛蟲有明顯sharp band

犬複殖器絛蟲 Dipylidium caninum

相關統計：以1999～2002 年德國流浪貓狗為例子

  • 狗感染寄生蟲比率較貓高（貓較愛乾淨，狗有扒糞的習慣）
  • 狗以感染線蟲（Nematodes）、絛蟲（Cestodes）、球蟲（Coccidia）居多

感染

  • 多數貓狗帶有犬複殖器絛蟲，應避免和貓狗親密接觸、玩親親
  • 定期替貓狗驅蟲以及身上的跳蚤

型態

  • 10～70 公分、乳白色
  • 額突上有1～7 個環鉤，越老的蟲，環的數目越多
  • 蟲體狹長，細的一端為頭節，粗的一端大片的是受孕體節

頭節 ( Scolex )

  • 0.37 × 0.35 mm
  • 有額突 ( Rostellum )：可自由伸縮，有很多圈
  • 4 個吸盤
  • 額頭上有1～7 個吻鉤 ( Hooklets )，越老的蟲，吻鉤的環數越多

成熟體節 ( Mature Proglottids )

  • 雙套生殖器，生殖孔有兩個 ( 教過的絛蟲中唯一 )
  • 約有200 個睪丸

受孕體節 ( Gravid Proglottids )

  • 子宮分化成許多卵囊 ( Egg Capsule )，生殖器官退化 ( 其他絛蟲大多可見子宮 )
  • 很狹長，呈「瓜仁」片狀
  • 切片可見卵囊，卵囊中卵的數目數顆至數十顆不等，卵的大小也不一

蟲卵

  • 中小型蟲卵35～60 μm，圓形
  • 每個Capsule 中含8～15 個蟲卵
  • 卵囊有大有小，囊內卵數與卵大小也有極大的差異

生命週期

  1. 中間宿主：狗蚤、貓蚤、狗蝨
  2. 保蟲宿主：貓、狗、狐狸等
  3. 感染型：心型擬囊尾蚴 ( Cysticercoid )
  4. 人是偶然宿主，通常為意外感染
  5. 感染過程
    1. 成蟲受孕體節自宿主糞便排出，受孕體節中的卵囊被釋放
    2. 中間宿主 ( 跳蚤等 )食入卵囊
    3. 蟲卵孵化成六鉤幼蟲 ( Oncospheres )
    4. 幼蟲在中間宿主體內發育為具感染力的擬囊尾蚴 ( Cysticercoid )
    5. 跳蚤成蟲 ( 含Cysticercoid )被貓、狗、狐狸等吃掉，完成生活史
    6. 人和貓狗親密接觸時可能意外吃下中間宿主，成為偶然最終宿主 ( 在人體內能發育成成蟲 )

分布

全世界性分佈

臨床症狀

  • 少量感染無症狀
  • 數量多時，會使腸胃不適、消化不良、沒有食慾、肛門搔癢 ( 移行時受孕體節脫落造成 )

治療

Praziquantel 配合瀉藥，以排出蟲體

診斷
