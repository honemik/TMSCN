# 阿米巴原蟲 

## 分類
- 致病性阿米巴 ( Pathogenic Amoeba )：會造成人體疾病，只有痢疾阿米巴一種 
- 自由生活阿米巴 ( Free-living amoeba )：還沒歸類成致病型阿米巴，平常不需要寄生，近幾年發現在偶然的機會下感染，會造成阿米巴腦炎，致死率有九成五 
- 共生性阿米巴 ( Commensal Amoeba )：腸道共生菌之一，不會有症狀，免疫低下時可能造成疾病 
- 嗜糞性阿米巴 ( Coprozoic Amoeba )：寄生在自然界腐爛的有機質上面 

## 型態 

### 囊體 Cyst

囊體內可以看到似染小體 ( Chromatoid Bodies and Bars )，與營養儲存有關，不同阿米巴有不同形狀的Chromatoid Bodies，是診斷上分辨阿米巴的依據

### 滋養體/活動體/阿米巴體 Trophozoite/Amoeboid  
- 型態不規則，會攝食跟運動 
- 只有一個細胞核，核膜上有顆粒狀的周邊染色質 ( 此構造痢疾阿米巴有但其他蟲不一定有 )

# 痢疾阿米巴 Entamoeba histolytica
## 簡介 
- 唯一會致病的阿米巴類，寄生在大腸，造成痢疾阿米巴症 ( Amoebiasis ) 
- 第二類法定傳染病 
  - 通報定義：符合以下任一條件 
    - 腹瀉，糞便中帶粘液、血絲、裡急後重、間歇性下痢，腹痛、發燒、噁心、嘔吐
    - 糞便檢體鏡檢發現疑似痢疾阿米巴之囊體或活動體 
    - 腸道及腸外之組織切片或潰瘍刮除發現痢疾阿米巴活動體者 
    - 超音波或電腦斷層診斷為肝膿瘍且血清抗體呈陽性反應 
  - 通報時限：24小時內
  - 通報後居家隔離或住院治療 
  - 疫情調查：訪視、填寫疫調單、家屬及接觸者實施糞便檢查 
- 西元1875年在俄羅斯的農夫身上首次發現 D. 食用被囊體汙染的食物和水感染，為糞口傳染 
- 可在20～43℃生存，最適溫度為37℃ ( 比較：自由生活型阿米巴可在溫泉生存，較耐熱 ) 

## 流行病學
- 每年約有5000萬人感染 
- 在熱帶、副熱帶地區或是密集、衛生條件不佳區域 ( 孤兒院、監獄 )較易發生感染 
- 台灣感染案例多來自外籍移工或是山區原住民 ( 飲泉水、生食 ) 

## 形態 

### 滋養體：

12～60 μm、單核，左下圖可見其中有多個被吃掉的紅血球，細胞核有周邊染色體 

### 囊體
- 10～20 μm、4個核、有似染小體 ( Chromatoidal Bar ) 
- 右下圖為發育中的囊體，有3個核，正中央有棒狀似染小體 

## 生活史
感染階段：囊體
吃進去的囊體會在腸道分裂出8個滋養體， ( 4個核變4個滋養體，每個滋養體再二分裂一次 )
滋養體會在大腸中持續進行分裂生殖，過程中有些滋養體會形成囊體從糞便排出

## 症狀及致病機轉 

### 無症狀、自限性感染 ( 90% ) 

- 痢疾阿米巴沒有破壞腸道黏膜層，但仍會產生囊體從糞便排出 
- 長期無症狀感染者會成為囊體傳播者 ( Cyst Passer )
-  
### 阿米巴腸炎 Amoebic Colitis  ( 10% ) 
- 致病機轉 
  - 溶解破壞腸道黏膜上皮細胞 ( MEC, Mucosal Epithelial Cell ) 
  - 利用Galactose Lectin和GalNAc ( N-acetyl-D-galactosamine Lectin ) 黏附宿主細胞 
  - 右圖是體外培養的痢疾阿米巴吞噬倉鼠卵巢上皮細胞的TEM圖， 可見痢疾阿米巴還沒消化完被吞噬的上皮細胞就吞噬下一個 
- 症狀 
  - 痢疾 ( Dysentery )：腹痛、腹脹 ( Distension )、低燒、血便、裡急後重 ( Tenesmus )，長期可能造成體重減輕 
  - 錐形瓶狀潰瘍 ( Flask-shaped Ulcer )：因阿米巴攝食紅血球 ( 左下圖 )
  - 腸穿孔 ( Perforation )：出現在嚴重感染 ( 缺乏治療、免疫力弱 )上，可能進而造成腹膜炎或二次細菌感染，致死率高 
  - 阿米巴肉芽腫 ( Amoeboma/Amebic Granuloma )：慢性感染，常被誤認為癌症 ( 右下圖 ) 
    - 糞便潛血：不是所有患者都能看到糞便中有血，但檢驗通常呈陽性 ( Heme-positive ) 
- 病程：錐形瓶狀潰瘍→嚴重潰瘍，絨毛脫落 → 腸穿孔、二次細菌感染 → 阿米巴肉芽腫 

### 腸道外阿米巴症  Extra-intestinal Amoebiasis  ( ＜1% )

途徑
- 直接穿透
  - 滋養體直接穿過腸道抵達其他器官 ( 腸穿孔 )
  - 通常抵達肝，再從肝穿過橫膈膜至肺 ( 肝和肺是最常發生腸道外阿米巴症的器官 ) 
- 血液循環：滋養體在腸道進入血液循環，透過血液抵達其他器官 

症狀 
- 肝膿瘍 ( Liver Abscess ) 
  - 通常在右下葉 
  - 內部充滿組織液
  - 周圍可能發現囊體 
  - 下左圖為肝膿瘍患者的X光圖， 可見右側橫膈膜升高；下右圖將膿瘍處理後可見空洞 
- 肺：直接穿透通常在右下葉，透過血液循環則不一定 
- 皮下、生殖道、腦：少見 

### 診斷

糞便檢查 
- 從形態學上難以辨別histolytica和在腸道共生非致病性的E. dispar、E. moshkovski，需仰賴較精準的分子生物學診斷 
- 腸道活躍度高會在糞便中看到滋養體，活躍度低 ( 便秘 )則看到囊體 

血清學：針對GalNAc Lectin開發單株抗體 ( Monoclonal )，以ELISA分析，只有感染痢疾阿米巴會呈陽性，臨床上很精準的方式但相對昂貴 

分子生物學 
- 通常是實驗室在用，可精準分辨是哪一種阿米巴，臨床不常用 
- Nested PCR：測16S-like rRNA的大小，可分辨E. histolytica、E. dispar和E. moshkovski D. 腸道外阿米巴症的診斷 
- 影像學輔助 
  - 放射線攝影 ( Radiography )可看到肝臟腫大、橫膈膜升高 
  - CT可看出膿瘍 
- 抽液：抽取肝膿瘍內的液體檢測是否有病原存在，通常只會有囊體不會有滋養體 

## 治療 

Metronidazole 
- 抑制DNA合成 
- 主要殺死滋養體，對囊體效果差 
- 口服，一天三次，治療十天 
- 孕期最初三個月禁用，用Tinidazole、Nitazoxanide代替 
- 原蟲類最常使用的藥物，阿米巴類和許多其他腸腔道內的原蟲都適用 

腸道製劑 ( Luminal Agent ) 
- 殺死囊體，與Metronidazole搭配使用 
- Iodoquinol、Diloxanide ( 最常用 )、Paromomycin ( 抗生素，感染過程中可能伴隨細菌感染 ) 

## 預防 
- 食物、飲水要乾淨 
- 飲用溪水前先用碘殺菌 ( Iodine Treatment ) - 不要使用糞肥 ( Feces Fertilizer )
- 確認廚師等食物工作者沒有感染 
- 避免口交、肛交等性行為

# 梨形鞭毛蟲 Giardia lamblia 

- 又稱G. intestinalis、G. duodenalis，中國譯名為蘭氏賈第鞭毛蟲 
- 早在1681年就由雷文霍克在其糞便中發現 
- 為造成旅行者腹瀉/水土不服 ( Traveler’s diarrhea )的原因之一 
- 對氯有抗性，為避免感染自來水還是建議煮過再喝 
  

## 預防

避免飲用與食用受汙染的水與食物

## 形態 

Cyst 
- 8～14μm、橢圓形 
- 具有4個細胞核、核無Peripheral Chromatin I
- 細胞中間具有中柱 ( Median Bodies )，用以生發鞭毛 

Trophozoite
- 9～21 μm、西洋梨形
- 兩個核、四對鞭毛 ( 前、側、腹、尾 )
- 腹面有吸盤，可以吸附在腸道上皮細胞
- 移動速度快但不激烈，像掉落的樹葉般移動 ( Falling Leaf Movement )

## 生活史

1. Cyst ( 感染型 )隨食物或水源進入人體
2. 到小腸中變為Trophozoite，之後採用二分裂法進行無性生殖
3. 一部份Trophozoite 會轉變成囊體又隨糞便排出 ( Trophozoite 和Cyst 都可以在糞便中發現 )
4. 在腸道中剛發育成Trophozoite 時，蟲體表面會有高度變異性的抗原 ( VSGs )脫逃免疫反應

## 症狀
- 有1～2 周潛伏期
- 免疫力強的人多半為無症狀，只為攜帶傳染源的Cyst Passers

急性期
- 持續腹瀉、胃痙攣、嘔吐
- 阻礙小腸吸收脂肪，造成病患糞便顏色淺、富含油脂且具惡臭
- 脫水

慢性期
- 在未開發國家較常見，因無法立即就醫而長期感染
- 體重下降、小腸絨毛變平坦 ( 下圖右 )、營養流失 ( 無法吸收脂肪、乳糖、Vitamin A、B12 )

## 診斷
- 糞便檢測：主要使用來確診的方法，囊體的排放有三種類型
  - 囊體以極多的量排出
  - 只有部分糞便有囊體存在
  - 以上兩者的混合型 ( 連續幾星期有大量囊體排出，中間有一小段極少囊體排出的間隔 )
  - 因此採檢體要在三個不連續的天數進行才能確診
- 免疫檢定方法：如ELISA、IFA ( 免疫螢光染色 )

## 流行病學
- 是美國發生率最高的寄生蟲疾病 ( 跟生飲自來水有關 )
- 高風險族群：5 歲以下孩童及孩童母親
- 在自然界中有許多保蟲宿主，以河狸 ( Beaver )最重要，因其會在水庫上游築巢，把Cyst 帶進民生用水中，造成感染

## 治療及預防
- Metronidazole ( Flagyl )：首選
- Tinidazole、Nitazoxanide
- 如遇到抗藥性需混和藥物治療和延長治療時間
 

# 陰道滴蟲 Trichomonas vaginalis

## 簡介
- 透過性行為傳染 ( STD, Sexually Transmitted Disease )，因此沒有Cyst 階段，只有Trophozoite
- 感染女性會造成生殖道病變、早產、新生兒體重過低；男性約75%無症狀
- 會增加感染HIV 的風險

## 預防
- 正確使用保險套
- 在公共場合如游泳池、溫泉、三溫暖等留意衛生。陰道滴蟲是主要透過性接觸傳染，而少數三溫暖間接感染的案例得知， 當外界的溫度濕度較高時存活較久，仍有可能透過椅子傳染

## 型態
- 7～30μm、單核
- 5 根鞭毛 ( 4 前1 尾 )
- 有波動膜 ( Undulating Membrane )，長度只有身體的一半
- 運動方式：激烈、無方向性

## 生活史
- 經由性接觸傳播
- 在生殖道靠二分裂法複製繁
- 也有泡三溫暖中標的案例，表示其 Trophozoite可在適當環境中短暫存活

- 頻尿與排疼痛
- 較高濃度的 Estrogen具有保護效果，因此停經婦女時會建議服用 具有保護效果，因此停經婦女時會建議服用 Estrogen降低感染風險
- 男性少有症狀、罕見情況下會感染前列腺、精囊、泌尿生殖道

## 診斷
- Wet-mount Slide：看生殖道的分泌液有無蟲體存在  ( 需：看生殖道的分泌液有無蟲體存在  ( 需10分鐘內完成，不然蟲體會死亡 )
- Papanicolaou ( Pap )Smear：子宮頸抹片
- 血清學檢查： EIA、Direct Fluorescent Antibody ( DFA )、 Latex agglutination ( LA )
- PCR 定序

# 大腸纖毛蟲 Balantidium coli

## 簡介
- 唯一會致病的纖毛蟲類
- 有雙核 ( 1 大1 小 )，在Cyst 和Trophozoite 都可見
- 不常見、研究少

## 形態

Cyst
- 50～70μm、圓形
- 具有2 個細胞核 ( 1 大1 小 )


Trophozoite 
- 有兩種大小 ( 只有大小差異，生理功能沒差 )
  - 小：42～60μm
  - 大：90～120μm
- 具有2 個細胞核 ( 1 大1 小 )
  - 小核 ( Micronucleus )：與基因和細胞複製有關
  - 大核 ( Macronucleus )：與營養代謝有關
- 有胞口 ( Cytostome )用以攝食

## 生活史
- Cyst ( 感染型 )隨食物或水源進入人體
- 到大腸中變為Trophozoite，之後採用二分裂法進行無性生殖
- 一部份Trophozoite 會轉變成囊體又隨糞便排出
- 纖毛蟲也可以行有性生殖 ( 接合生殖 )
## 病理與症狀
- 多數人無症狀
- 造成輕微腸炎、火山狀的大腸潰瘍
- 以透明質酸酶 ( Hyaluronidase )侵犯腸道組織，造成更容易感染其他細菌及HIV

## 診斷
- 糞便檢查
- 因病例少尚無血清學檢查

## 治療：
- Tertracyclin：首選藥物
- Metronidazole、Iodoquinol、Paromomycin、Nitazoxanide

# 人芽囊原蟲 Blastocystis hominis

## 簡介
- 寄生於腸道
- 是否具致病性仍然有爭議
- 很常在檢體中發現

## 形態

1. 液泡狀  ( Vacuolar )
   - 最常見
   - 液泡占滿整個細胞、其他器被擠到邊緣
2. 顆粒狀  ( Granular )
3. 囊體狀  ( Cyst )
4. 阿米巴體狀  ( Amoeboid )

## 生活史

- 感染型： Thick-walled Cyst ( 在外界生存 )，經由攝食跑到小腸感染上皮細胞  ( 在外界生存 )，經由攝食跑到小腸感染上皮細胞 Thick-walled Cyst在 小腸破裂，釋放很多 Vacuolar Form蟲體
- Vacuolar Form蟲體有兩種成長路徑 蟲體有兩種成長路徑
  - Multi-vacuolar Form → 變成 Thin-walled Cyst → 破裂釋放更多 Vacuolar Form蟲體
  - 變成 Amoeboid Form → 變成 Thick-walled Cyst → 排出體外

## 症狀
- 通常無症狀、即使有也能靠自體免疫復原
- 造成拉肚子、脹氣，免疫力低下可能會水瀉脫水

## 診斷

糞便檢查有無 Cyst-like的蟲體

## 治療 

尚無標準用藥 、依照慣例原蟲就用 Metronidazole
