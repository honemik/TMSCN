# 薑片蟲 Fasciolopsis buski

## 感染

接觸被感染的扁捲螺或水生植物

## 型態

- 大小: 3~7 cm x 1~2 cm x 3~7 cm
- 人體最大的寄生蟲
- 腸管在兩個吸盤 ( 口、腹 ) 之間分叉後延蟲體兩側伸至尾端
- 蟲體一端具有口吸盤 ( Oral Sucker ) 和大五倍的腹吸盤 ( Acetabulum ) 
- 蟲體後半幾乎由高度分支的睪丸佈滿；卵巢有向外少許，由此至腹吸盤之間纏繞著子宮

## 生命週期

1. 蟲卵在溫暖的水中，經過三週孵化成纖毛幼蟲 ( Miracidium )、破除卵蓋 
2. 感染扁捲螺 ( 第一中間宿主 ) ，並在其體內發育為胞蚴 ( Sporocyst )、雷蚴 ( Redia )、尾蚴 ( Cercaria )，耗時約一個半月
3. 漂浮在水面，或游至菱角、荸薺、筊白筍、蓮藕等水生植物 ( 第二中間宿主 ) 附著，發育成為囊蚴 ( Metacercaria )
4. 人、豬  ( 最終宿主 ) 攝取這些植物或飲用生水，囊蚴便可藉助消化道分泌物分解外囊，並吸附
在黏膜上，經歷 1 ~ 3 個月便可發育成蟲並產下後代


## 分布

中國、臺灣 ( 現在幾乎沒有了 )、東南亞

## 臨床症狀

主要寄生在小腸，數量多時可達上千隻甚至能擴散胃和大腸，一般吸附在腸黏膜上，以血液維生，患者會有腹痛、便秘、腹瀉、貧血等症狀交替發生，也可能會有出血 ( Hemorrhage )、潰瘍 ( Ulceration )、潰瘍 ( Ulceration )、膿腫  ( Abscess )等現象，不過多數患者幾乎都不會出現 明顯症狀

感染症狀程度
- 輕度：無症狀 ( Asymptomatic )
- 中度：腹痛、便秘瀉貧血
- 重度：腸阻塞 ( Bowel Obstruction )
- 大約 40% 的患者 ( ＞30% ) 會有嗜酸性球增多症 ( Eosinophilia )

## 治療

Praziquantel 25 mg/kg，一天三次

Praziquantel為對抗吸蟲的萬用藥，幾乎所有吸蟲病皆可治療

## 診斷
- 成蟲每天產出超過 25000 顆卵
- 大小約 130~140 μm x 80~85 μm
- 淡黃色，均勻卵圓形
- 卵蓋 ( Operculum ) 較小且不明顯，卵殼薄而透

- 以糞便中的蟲卵，或是排出、吐成來區分其與細菌性腸炎阿米巴原感染
- 從睪丸的形狀、兩個吸盤比例頭錐態來區分其與牛羊肝蟲 ( Fasciola hepatica  )
- 若難以用蟲卵 分辨薑片蟲和牛羊肝吸，臨床症狀、旅遊史吃過食物接觸都能協助

# 橫川異形吸蟲 Metagonimus yokogawai

## 感染

吃入未煮熟的魚裡的囊蚴
預防：避免食用生魚片或醃製的魚乾 ( 醃製無法殺死寄生蟲 )

## 型態

- 大小：1 ~ 2.5mm × 0.3 ~ 0.4 mm
- 腹吸盤 ( GA, Gentioacetabulum )位於中線右側
- 無生殖吸盤 ( Genital Sucker )，兩個睪丸 ( TE )位於蟲體後端，呈 45°弧線
- 卵黃腺 ( Yolk Gland )分佈在後 1/3兩側

## 生命週期

1. 第一中間宿主：淡水螺類 ( 海蜷、燒酒螺、栓海螺 )。蟲卵在螺體內發育經過胞蚴、雷蚴階段，變成尾蚴後逸出
2. 第二中間宿主 ：魚和蛙 。尾蚴侵入後在宿主體內發育成囊蚴
3. 最終宿主：海鳥、貓、狗、人
   1. 成蟲寄生於哺乳動物的腸道
   2. 如人類吃未煮過的魚，可能不慎攝入藏在鱗片下的囊蚴，囊蚴被吞食後發育為成蟲，並在小腸空腸 ( Jejunum )、迴腸 ( Ileum ) 上段寄生，治療過後才會在糞便中發現

## 分布

分布於遠東、西伯利亞巴爾幹半島、西班牙

## 臨床症狀

- 輕微感染：無症狀 ( Asymptomatic )
- 重度感染：慢性黏膜下痢 ( Chronic Mucous Diarrhea )、腹痛等消化道症狀、消瘦、肉芽腫
- 由於成蟲體型小，在腸道寄生時可能鑽入腸壁，因而蟲卵可進入腸壁血管並隨血流到達腦、脊髓、肝、脾、肺、心肌等組織或器官，造成嚴重後果

## 治療

治療：Praziquantel 

## 診斷

- 28 ~ 32 μm × 14 ~ 18 μm
- 黃褐色至褐色的橢圓或卵形
- 卵殼厚、表面不光滑，內含明顯之纖毛幼蟲

常規的病原學檢查方法是糞便塗片及沉渣鏡檢蟲卵 

各種異形吸蟲卵形態相似，且難以鑒別同為小型蟲卵的中華肝吸蟲 ( C. sinensis )，因此了解
該地區是否有異形吸蟲的存在，將有助於診斷，若能獲得成蟲，則可根據成蟲的形態進行判斷

# 異形異形吸蟲 Heterophyes heterophyes

預防：避免食用生魚片或醃製的魚乾 ( 醃製無法殺死寄生蟲 )

1. 第一中間宿主：淡水螺類 ( 海蜷、燒酒螺、栓海螺 )。蟲卵在螺體內發育經過胞蚴、雷蚴階段，變成尾蚴後逸出
2. 第二中間宿主：烏魚、海水魚。尾蚴侵入後在宿主體內發育成囊蚴
3. 最終宿主：海鳥、貓、狗、人
   1. 成蟲寄生於哺乳動物的腸道

## 分布

中東、南歐、亞洲

## 臨床症狀

- 輕微感染：無症狀 ( Asymptomatic )
- 重度感染：慢性黏膜下痢 ( Chronic Mucous Diarrhea )、腹痛等消化道症狀、消瘦、肉芽腫
- 由於成蟲體型小，在腸道寄生時可能鑽入腸壁，因而蟲卵可進入腸壁血管並隨血流到達腦、脊髓、肝、脾、肺、心肌等組織或器官，造成嚴重後果

## 治療

治療：Praziquantel 

## 診斷

- 23 ~ 30 μm × 14 ~ 17 μm
- 黃棕色或褐的橢圓至卵形、無肩峰
- 卵蓋對側殼厚、有小棘，內含明顯之纖毛幼蟲

# 棘口吸蟲 Echinostoma spp.

棘口屬的都算 ( E. hortense, E. trivolvis, E. macrorchis, E. revolutum sensu lato, E. ilocanum, E. cinetorchis, E. echinatum [= lindoense] and E. fujianensis )

## 感染

誤食含囊蚴的中間宿主 ( 螺螄或雙殼貝類體內，甚至在植物上結囊；常見的包括蝸牛、水生植物、昆蟲、青蛙、魚 )，成蟲寄生於小腸

## 型態

- 大小：2.5 ~ 6.5 mm × 1 ~ 1.35 mm
- 外觀細長，腹吸盤比口吸盤大3 ~ 4 倍 
- 馬蹄型頭部 ( Horseshoe-shaped Collar )，口吸盤背側及外側有兩列頭棘，體表有皮棘 ( Spine )
- 二個睪丸前後排列於蟲體後端 
- 卵黃腺分佈蟲體後 2/3 兩側

## 生命週期

1. 第一中間宿主為淡水螺類，纖毛蚴侵入螺體後經胞蚴和兩代雷蚴階段後發育成尾蚴 
2. 第二中間宿主種類多元，尾蚴可在子雷蚴體內結囊、逸出後在原來的螺體內結囊、侵入其他
螺螄或雙殼貝類體內，甚至在植物上結囊；常見的包括蝸牛、水生植物、昆蟲、青蛙、魚 
3. 最終宿主主要是鳥禽類，其次是哺乳類，爬行類，少數寄生於魚類；有的可在多種宿主寄生 
4. 一般不以人為最終宿主，被感染幾乎都是因誤食含囊蚴的中間宿主，成蟲寄生於小腸 

## 分布

1. 常見E. ilocanum 分布於菲律賓和泰國；E. malayanum 和E. revolutum 分布於泰國 
2. 在菲律賓呂宋島北部，E. malayanum 的感染率平均佔調查人口的10％，最高可以超過40％ 
3. 在泰國北部，寄生於人體的Echinostomes 盛行率高達50%

## 臨床症狀

1. 輕微感染：一般無症狀 ( Symptomless )，時有輕微腸胃不適如腸胃脹氣 ( Flatulence )、腹瀉 ( Diarrhea )、便秘 ( Constipation ) 
2. 嚴重感染 
   1. 成蟲寄生於小腸上段絨毛間，以頭部插入黏膜並釋放毒素，發炎後引起全身性毒性反應 
   2. 可能出現腹痛、腹瀉等腸胃道症狀；更嚴重者會厭食、下肢浮腫、貧血、消瘦、發育不良

## 治療

治療：Praziquantel

## 診斷

常用的糞便檢查方法 ( Fecal Examination )：直接塗片法、沈澱法等，多種棘口吸蟲的卵在形態上都很相似

# 人雙口吸蟲 Gastrodiscoides hominis

## 感染

當人生食水生植物 ( 如菱角 ) 時，食入表皮上的Metacercaria 而被感染 

## 型態

- 大小：5 ~ 7 mm × 2 ~ 4 mm 
- 紅棕色角錐形蟲體，表面光滑 
- 口吸盤小，呈球狀 
- 有一明顯凹陷及背面突起 
- 睪丸大、分葉、前後排列

## 生命週期

與薑片蟲類似之生活史，寄生於大腸壁

1. 蟲卵在溫暖的水中，經過三週孵化成纖毛幼蟲 ( Miracidium )、破除卵蓋 
2. 感染扁捲螺 ( 第一中間宿主 ) ，並在其體內發育為胞蚴 ( Sporocyst )、雷蚴 ( Redia )、尾蚴 ( Cercaria )，耗時約一個半月
3. 漂浮在水面，或游至菱角、荸薺、筊白筍、蓮藕等水生植物 ( 第二中間宿主 ) 附著，發育成為囊蚴 ( Metacercaria )
4. 人、豬、鹿 ( 最終宿主 ) 攝取這些植物或飲用生水，囊蚴便可藉助消化道分泌物分解外囊，並吸附在黏膜上，經歷 1 ~ 3 個月便可發育成蟲並產下後代

## 分布

分布於馬來西亞、緬甸、越南南方、印度阿薩姆 ( 1876 年T.R. Lewis & J. McConnell
最早發現 )、圭亞那、俄羅斯的伏爾加三角洲

## 臨床症狀

一般無症狀有時會伴隨腸道問題：腹瀉、發燒、腹痛、腹絞痛 ( Colic )、黏液分泌增加

## 治療

- 傳統療法：肥皂水灌腸 ( Soap Enema )
- Praziquantel：一日三碇 ( 25 mg/kg )
- Tetrachloroethylene：空腹吃 ( 0.1 mg/kg )，副作用多且會致癌，目前已停用

## 診斷

- 大小：145 ~ 170 μm × 60 ~ 75 μm，屬於大型蟲卵 
- 淡黃色菱形、前端狹窄，形態接近薑片蟲卵，但薑片蟲為卵圓形 
- 卵蓋細小 
- 卵排出時未發育成胚，內含一個卵細胞和許多卵黃細胞

# 衛氏肺吸蟲 Paragonimus westermani

## 感染

若人類、貓、狗吃下未完全處理乾淨的甲殼類動物 ( 醉蝦、醉蟹 )，會有染病風險

## 型態

- 大小：6 mm × 10 mm 
- 體型卵圓、色澤棕紅，與咖啡豆相似 
- 兩個狀似銀杏葉狀的睪丸並列於蟲體後三分之一處，因此分類於並殖科 
- 表皮有許多棘刺，可幫助肺吸蟲勾在肺部 
- 口吸盤與腹吸盤約等大，分別位於前端和腹部中央

## 生命週期

1. 第一中間宿主：自由川蜷 ( Semisulcospira libertina ) 被纖毛幼蟲 ( Miracidium ) 感染 
2. 第二中間宿主：淡水甲殼綱動物 ( 蝦、蟹 ) 被尾動幼蟲 ( Cercaria ) 感染，並於鰓或肌肉組織形成囊腫 ( Cyst ) ，發育為囊狀幼蟲 ( Metacercaria )  
3. 最終宿主：生食第二中間宿主而感染 
4. 未成熟吸蟲侵犯腸胃道，並循腹腔穿過橫膈膜到肺部發育成蟲 
5. 蟲體會移行至各處異位寄生 ( Ectopic Paragonimiasis ) ，造成其他器官損害 

## 分布

在全世界研究最多、分布廣泛的肺吸蟲，主要盛行於東南亞、韓國、中國、日本、新幾內亞

台灣因河川汙染嚴重，居民較少食用淡水蝦蟹，儘管有衛氏肺吸蟲分布，但少有此類病例

## 臨床症狀

- 咳嗽、呼吸困難、血痰、胸腹疼痛、腹瀉等症狀
- 痰中常會含少許血液並包含蟲卵或Charcot-Leyden Crystals 結晶
- 血液中嗜酸性球 ( Eosinophil ) 大量增加，相關抗體也會增高 
- X 光照起來像肺結核 ( 因咳嗽咳得很嚴重，常誤診為肺結核 )

病程分期 
- 急性期：童蟲 ( Schistosomulum ) 移行，幼蟲進入腹腔，穿過橫膈移行至肺 
- 慢性期 ( Chronic Stage )：成蟲肺部寄生 
- 浸潤期 ( Stage of Abscess )：也稱膿腫期，引起發膿發腫 
- 囊腫期 ( Stage of Fibrous Cyst )：形成囊腫，免疫細胞將蟲體包起來成囊 
- 纖維疤痕期 ( Stage of Fibroscar )：囊腫纖維化

異位寄生 ( Ectopic infections )
- 吸蟲除了跑到肺部寄生，也可能移行至其他部位，尤其是腦部寄生最為嚴重 
- 如寄生於腦部，常產生空間占據性病灶，造成肢體障礙、腦性麻痺等症狀，也經常發生膿瘍 
- 幼兒與兒童最易發生。有 90% 腦部異位寄生的個案在 30 歲以下

## 治療

Praziquantel：25 mg/kg，一天三次，共2 天 ( t.i.d.  ×  2 day )

Bithionol：15 ~ 25 mg/kg，一天兩次，共10 ~ 15 天 ( b.i.d.×10 ~ 15 day )

## 診斷

- 大小：68 ~ 118 μm x 9 ~ 67 μm 
- 卵呈黃褐色
- 蟲卵具有厚壁，一端較平使得外觀不對稱 
- 較大的一端則有清楚可見的卵蓋，另一端為較厚實的底部

- 痰、糞蟲卵檢查：痰液、糞便、胃液中的蟲卵為鑑別診斷 ( Differential Diagnosis ) 的依據 
- 免疫學檢查 
  - 補體結合檢測 
  - 皮內試驗 ( Skin Test ) ：敏感度高 ( 80% ~ 90% ) ，但在治療後10 ~ 20 年抗體反應仍維持陽性
  - ELISA 
  - 單抗檢測抗原
