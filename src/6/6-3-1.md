| 病毒 | 正黏病毒 | 副黏病毒 |
| --- | --- | --- |
| 病毒結構 | 螺旋體，具外膜 | 螺旋體，具外膜 |
| 基因組 | ssRNA | ssRNA |
| RNA 複製處 | 細胞核 | 細胞質 |
| RNA 分段 | 分為7～8 段 | 不分段 |
| 基因重組 | 經常 | 無 |

# 正黏液病毒

## 特徵
- 有Influenza A, B, C 三類。
- 具有套膜 ( Envelope )，可被肥皂或清潔劑 ( detergent )去活性。
- 遺傳物質為分段負義線性單股RNA ( segmented negative-sense RNA genome )。 
- 分段分成八段 ( Influenza A, B )，但Influenza C 例外只有七段。 
- 因為遺傳物質是RNA，所以基因不穩定，容易突變造成疾病流行。 
  - Annual epidemics：是基因產生點突變 ( mutation：drift )，例如每一年不同型的流感。 ( drift 是小突變 ) 
  - Periodic pandemics：是RNA segment 彼此排列組合 ( reassortment：shift )，有可能和不同種病毒混合，常造成大流行。 ( shift 是大突變 )
- 只有Influenza A 是zoonosis ( 人畜共通 )，Influenza B ( 較不易從動物傳染，Influenza C ( 動物也有 )
- Influenza 的RNA 在細胞核裡複製。 ( 大部分RNA 病毒都在細胞質複製 )
- 病毒結構為螺旋形，外有套膜。

## RNA轉譯出的蛋白質

第1~3 段RNA 轉譯出病毒自己攜帶的RNA polymerase ( RNAP )，分別是PB2, PB1, PA，對RNA 複製時的轉錄很重要。 

Hemagglutinin ( HA ) 
- 是trimer 的構造，為第四段RNA 轉譯出的蛋白質。 ( NS2 運送 ) 
- 有18 種 ( H1~18 )，前幾種會感染人。 
- Viral attachment protein：和上皮細胞膜上受器的sialic acid 結合。 
- 促進宿主細胞膜與套膜的融合。 ( 酸性環境下 ) 
- 使紅血球凝集 ( 人、雞、天竺鼠 )。 
- 與中和性抗體 ( Neutralizing antibody )產生反應。 ( 疫苗成份之一 ) 

第五段RNA 轉譯出NP ( nucleoprotein, nucleocapsid protein )，會和轉錄出的RNA 結合，可用來分辨是哪一種Influenza ( A, B, C )。 

Neuraminidase ( NA ) 
- 是tetramer 的構造，為第六段RNA 轉譯出的蛋白質。 
- 有11 種 ( N1~11 )。  
- 有酵素活性，具有分解能力。 
- 切割醣蛋白上的sialic acid ( 整個受器被切下來 )，避免病毒被宿主細胞受器的sialic acid黏住，促進病毒的釋放。 
- 是兩種抗病毒藥的作用目標：zanamivir ( Relenza )、 oseltamivir ( Tamiflu )

第七段RNA 轉譯出的蛋白質是M1, M2，可用來分辨是哪一種Influenza ( A, B, C )。 ( 利用 Alternative splicing 轉譯出不同蛋白質 ) 
- M1 ( Matrix protein )：結構蛋白，和nucleocapsid, envelope 作用促進病毒的組裝。 
- M2 ( Membrane protein )：套膜上的蛋白，可以形成離子通道改變pH 值，並利用pH 值的調節促進病毒uncoating 以及HA 的折疊，影響HA 穩定性。

第八段RNA 轉譯出的蛋白質是NS1 和NS2，NS2 可以幫助複製完的病毒離開細胞核。 ( 利用Alternative splicing 轉譯出不同蛋白質 )

### 病毒的複製 

// 圖

1. 病毒用HA 和宿主細胞receptor 上的sialic acid 結合，宿主細胞會以endocytosis 的方式形成endosome 使病毒進入細胞。 
2. M2 作用產生酸性環境使病毒套膜與endosome 細胞膜融合，令病毒uncoating，釋放RNA。 
3. RNA segments 進入細胞核，用病毒自己的酵素轉錄出帶有CAP 和poly-A tail 的mRNA。 ( CAP 是病毒用自己的endonuclease 把宿主RNA 的CAP 剪下作為自己的primer ) 
4. 病毒的mRNA 在細胞質進行轉譯產生蛋白質。PB1, PB2, PA, NP, NS1, NS2 這些蛋白質 ( 構成nucleocapsid )會進入細胞核。 ( PB1, PB2, PA 稱為RNAP ) 
5. negative-sense RNA 會轉換成positive-sense 作為模板之後再進行複製，做出negative-sense RNA。 
6. negative-sense RNA 和進入細胞核的蛋白質結合形成viral ribonucleoprotein ( vRNP )，每一條vRNP 含有一條negative-sense RNA、NP、RNAP。 
7. NS2 把vRNP 送出細胞核進行組裝。 
8. M2 避免在高基氏體上的HA、NA 過度折疊，HA、NA 被送去鑲嵌在細胞膜上。 
9. 透過M1，nucleocapsid 和細胞膜結合形成envelope。 (  envelope 上可見HA, NA, M1, M2 ) 
10. 病毒以budding ( 出芽 )離開宿主細胞產生新的病毒。

### 致病機制及免疫反應

// 圖

1. 經由飛沫傳染 ( aerosol inoculation )進入並複製，攻擊黏膜、纖毛等上皮細胞－破壞初級防禦系統，導致細菌黏附感染產生肺炎。 
2. NA 把sialic acid 切除，使得病毒能進入組織。 
3. 病毒進入肺部促進細胞跟細胞之間的傳播 ( promote cell-to-cell spread )以及傳染到其他宿主 ( transmission )。  
4. 進入下呼吸道造成支氣管及肺泡上皮細胞脫落。 
5. 黏膜發炎造成黏膜下腫脹。 ( monocyte & lymphocyte ) 
6. 產生interferon ( 使症狀更嚴重 )和lymphokine 以及CMI response ( NK & T cell )，造成系統性反應，可能引起免疫風暴。 ( immunopathogenesis ) 
7. 上皮細胞的受損產生局部症狀。 
8. 抗體在預防感染上扮演重要角色。 

## 抗原性的改變 antigenic changes

### Antigenic drift 
- 是較為微小的改變 
- 關於HA 和NA 基因的突變 
- 約每2～3 年發生一次 
- 會造成地區性A 和B 型流感的爆發 

### Antigenic shift 
- 是較大的改變 
- 在不同品種間基因重組時產生 
- 跟大流行 ( pandemics )有關 
- 只會發生在A 型流感上

## 流感病毒的命名

命名規則：病毒種類／宿主 ( 如果是人則省略 )／發現地/實驗室編號／發現年分 ( HXNY )

通常只有Influenza A 有最後一項 ( H & N 分型 )，像是Influenza B 沒有Antigenic shift，故沒有H & N 分型


## 治療方法 

### 金剛烷胺 Amantadine 和雷曼他定 Rimantadine 

- 目標：作用在M2 protein 上，可阻止病毒從宿主細胞釋出 ( uncoating ) 
- 沒辦法影響 B 和 C 型流感

### Zanamivir Relenza 和Oseltamivir ( Tamiflu )
- 目標：作用在神經氨酸酶 ( neuraminidase )，可抑制其功能，進而預防被感染的細胞釋放病毒
-  能抑制B 和C 型流感 
-  在A 型流感疾病開始的前24－48 小時，對於治療是很有效的

### 接種疫苗 
-  最好的病毒控制方法是免疫系統 
-  建議老年人和慢性肺炎或心臟病患者施打疫苗

# 副黏液病毒
 
副黏病毒科的病毒會經由呼吸道途徑造成感染。外型比正黏病毒略大，是一種有envelope 的負鏈ssRNA 病毒，Capsid 呈螺旋對稱型，其RNA 複製和蛋白質合成都在細胞質內進行。RNA不分段，無基因重組現象。本科病毒感染細胞常引起cell-to-cell fusion ( 包括合胞體 ( syncytia )及多核巨細胞 ( multinucleated giant cells )的形成 )

|屬|病毒|
|---|---|
|副黏病毒屬 Paramyxoviruses|副流感病毒 Parainfluenza virus|
|副黏病毒屬 Paramyxoviruses|腮腺炎病毒 Mumps virus |
|麻疹病毒屬 Morbiliviruses|麻疹病毒 Mealses virus |
|肺炎病毒屬 Pneumoviruses |呼吸道融合病毒 Respiratory syncytial virus，RSV |
|肺炎病毒屬 Pneumoviruses |人類間質肺炎病毒 Human metapneumovirus，h-MPV |

另外，在1998 年因為馬來西亞及新加坡爆發嚴重腦炎，而確認的立百病毒 ( Nipah virus )及亨
德拉病毒 ( Hendra Virus，HeV )也屬於副黏液病毒。當時造成人類和其他動物 ( 尤其以豬隻為
主 )交叉感染，嚴重的則會引致死亡。

## 結構

### ( - ) ss RNA genome

### Helical nucleocapsid

- Nucleoprotein ( NP )：幫助維持基因體的結構。 
-  Polymerase phosphoprotein ( P )：促使RNA 的合成。 
-  Large ( L )protein：副黏液病毒的RNA polymerase。

### envelope

envelope 上帶有F protein 及Viral attachment protein ( 前者為較小的glycoprotein，後者為較大的glycoprotein )。 

- F ( fusion )protein：幫助病毒與宿主細胞融合 ( fusion )。另外，F protein 會引起cell-to-cell fusion，導致合胞體 ( syncytia )及多核巨細胞 ( multinucleated giant cells )的形成。
- Viral attachment protein：與宿主細胞上的receptor 結合，藉此依附在特定細胞表面，並引發replication。分為三種 ( HN、H、G )，而根據這三種glycoprotein 不同的活性，可以藉此分辨出副黏液病毒的不同屬 ( genera )。
  - Hemagglutinin-neuraminidase ( HN )：腮腺炎病毒、副流感病毒 
  - Hemagglutinin ( H )：麻疹病毒 
  - Glycoprotein ( G )：呼吸道融合病毒、人類間質肺炎病毒 

### M（matrix）protein

病毒envelope 與nucleocapsid 間的亞膜結構。

## Paramyxovirus 的複製

複製方式與其他負鏈的RNA病毒相似。轉錄、蛋白質合成、基因體複製皆位在宿主細胞的細胞質當中。

- 副黏液病毒利用viral attachment protein 和宿主細胞上的receptor 結合，接著以fusion 的方式（F protein 的幫忙）進入宿主細胞內。 
- 副黏液病毒的genome 先被轉錄成mRNAs，以及full-length（+）RNA template（需要L 及P protein 的幫忙）。 
- 轉錄過後的mRNAs 轉譯出病毒所需的蛋白質及醣蛋白；full-length（+）RNA template 則製造出病毒的genome（負鏈的ss RNA）。
- 新的genome 與蛋白質（L、P、NP protein）結合後，形成helical nucleocapsid。 
- 新形成的nucleocapsid 與M protein 及帶有病毒醣蛋白的細胞膜結合後，利用出芽的方式離開宿主細胞


# 麻疹病毒（Measles Virus）

麻疹病毒會引起麻疹（Measles，又稱rubeola）。（麻疹為其中一種典型會使小孩子起疹的疾病。其他包括風疹（Rubella，又稱德國麻疹、三日疹）、嬰兒玫瑰疹（Roseola infantum）、第五病（Fifth disease）、水痘（Chickenpox）。）

症狀明顯，主要是3C（咳嗽cough、結膜炎conjunctivitis、鼻炎coryza）、 1P（畏光photophobia）與Koplik’s spots。最嚴重的併發症是SSPE（亞急性硬化性泛腦炎）。

未有麻疹疫苗前，麻疹是一種普遍的兒童傳染病，常見於五歲以下的小朋友。雖然目前有麻疹疫苗，但在醫療尚未非常發達的國家，每年依然有許多案例的發生。

## 傳染特性

- 只有一種血清型。 
- 人是唯一宿主。 
- 透過呼吸道飛沫（respiratory droplets）傳染（例如：空氣、飛沫、或病人口鼻分泌物、或咽喉分泌物接觸傳染）。 
- 已知最具傳染力的疾病之一（attack rate＞90％）。 
- 好發於冬末及春季。 
- 得過麻疹後，獲得終身免疫。

## 致病機轉及免疫 

1. 感染呼吸道上皮細胞。 
2. 感染淋巴球，並經由血液造成全身性的感染（viremia）。  
3. 麻疹病毒會在結膜、呼吸道、泌尿道、淋巴系統、血管、中樞神經系統增生複製，造成相對應的疾病。 
4. 麻疹病毒所引起的紅疹（rash），是由於T 細胞攻擊受到病毒感染的血管上皮細胞所導致。 
5. 身體內藉由細胞免疫控制感染，並獲得終身免疫。 
6. 最嚴重的併發症是SSPE（亞急性硬化性泛腦炎），是一種progressive chronic 腦部神經退化性疾病。藉由Paramyxovirus 典型的cell fusion 手法在腦部神經細胞間擴散並造成損毀，可在數月至數年間導致coma & death。（ 105 共筆） 

## 流行病學
A.  麻疹病毒有大的病毒粒子（virion），容易被乾燥（dryness）和酸（acid）等因素所滅活。 
B.  麻疹病毒的傳染期（conjugation period）會比病徵（發病期）來得更早，所以很難在傳染期
出現前隔離病患以防止傳染。 
C.  傳染途徑：飛沫傳染。 
D.  誰容易得到麻疹： 
(1) 五歲以下幼童 
(2) 沒打過疫苗的人 
(3) 營養不良的人 
(4) 有免疫缺失的人 
E. 全世界都有麻疹病毒；常發生在秋冬（地方性的，endemic），尤其是人多壅擠的室內。 
F. 可以注射疫苗防止感染：MMR 是三種病毒合一的活性減毒疫苗（live attenuated vaccine），
包括麻疹病毒（Measles Virus）、腮腺炎病毒（Mumps Virus）及德國麻疹病毒（Rubella 
Virus）。
